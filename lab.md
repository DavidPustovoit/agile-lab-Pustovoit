## 5. Детальний опис User Story

Для деталізації обрано задачу з Епіку 2 (UI).

### Summary
Як користувач, я хочу бачити результат конвертації миттєво, щоб розуміти вартість валюти.

### Description
Користувач повинен мати можливість ввести суму в гривнях або іншій валюті, обрати цільову валюту, і система має автоматично розрахувати та показати кінцевий результат на основі актуального курсу.
Розрахунок має відбуватися або після натискання кнопки "Конвертувати", або в реальному часі при введенні.

### Acceptance Criteria (AC)
- Користувач може ввести суму в поле "Вихідна валюта"
- Система підтримує введення дробових чисел (через крапку або кому)
- При виборі валют (наприклад, USD -> UAH) відображається актуальний курс обміну
- Результат з'являється у полі "Результат" з точністю до 2 знаків після коми
- Якщо поле вводу порожнє, результат показує 0.00
- Якщо немає доступу до мережі, використовується останній збережений курс із кешу

### Definition of Done (DoD)
- Реалізовано UI компонент для виводу результату
- Написано Unit-тест для математичної функції конвертації
- Перевірено коректність відображення на мобільних пристроях
- Пройдено перевірку на введення великих чисел (мільйони)
