## 1. Створення Епіку
Для реалізації нових можливостей створено епік.

**Назва:** налаштування та персоналізація.

**Мета:** надати користувачу можливість адаптувати додаток під свої потреби (мова, сповіщення, валюти).

---

## 2. User Stories та Story Points
В рамках епіку створено 3 історії користувача. Оцінка складності виконана в Story Points (SP).

### Story 1. Зміна мови інтерфейсу
**Оцінка:** 3 SP
```
Як користувач, я хочу змінювати мову інтерфейсу, щоб користуватися додатком зрозумілою мовою.

**Acceptance Criteria**
* Є перемикач мови в налаштуваннях
* Тексти оновлюються миттєво без перезапуску
* Обрана мова зберігається після закриття додатку
```

### Story 2. Сповіщення про курс
**Оцінка:** 5 SP
```
Як трейдер, я хочу отримувати сповіщення при зміні курсу, щоб встигнути вигідно купити або продати валюту.

**Acceptance Criteria**
* Можливість увімкнути/вимкнути сповіщення
* Налаштування відсотка зміни (поріг срацьовування)
* Сповіщення приходять у фоновому режимі
```

### Story 3. Валюта за замовчуванням
**Оцінка:** 2 SP
```
Як користувач, я хочу обрати основну валюту, щоб вона завжди була першою у списку.

**Acceptance Criteria**
* Список доступних валют у налаштуваннях
* Обрана валюта закріплюється зверху списку конвертації
* За замовчуванням встановлено UAH
```

## 3. Subtasks (Підзадачі)
Для декомпозиції робіт кожну історію розбито на технічні підзадачі.

**Зміна мови інтерфейсу**
1. Підготувати файли локалізації
2. Реалізувати UI перемикача в налаштуваннях

**Сповіщення про курс**
1. Створити сервіс для фонових сповіщень
2. Додати екран налаштування відсотків

**Валюта за замовчуванням**
1. Реалізувати збереження вибору в пам'яті телефону
2. Змінити логіку сортування списку валют

## 4. Definition of Ready & Done
Вимоги до якості зафіксовано в коментарях до задачі "Зміна мови інтерфейсу".
```
Definition of Ready (DoR):
- Макет дизайну готовий
- Тексти перекладені

Definition of Done (DoD):
- Перемикання працює
- Немає багів
- Код у репозиторії
```
